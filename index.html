SQL · Database Analysis

Hospital Database Analysis

Advanced SQL querying on a relational hospital database to uncover physician workload patterns and patient risk factors.

The Goal

Write advanced SQL queries against a hospital database to answer real-world clinical questions: which doctors are overloaded relative to their specialty peers, how workload varies by month, and which patients should be flagged for additional monitoring based on age, allergies, and admission frequency.

What I Contributed

  • Engineered three advanced SQL queries joining multiple tables (doctors, patients, admissions) to extract physician workload trends and specialty-based comparisons
  • Built a correlated subquery to compare each doctor's patient count against the average for their specialty — identifying above-average workload physicians
  • Used strftime to extract month from admission dates and group workload by time period
  • Designed a multi-condition patient risk classification system using nested CASE statements, flagging patients by age group, allergy status, and admission frequency

Skills Gained

  • Advanced SQL: multi-table JOINs, correlated subqueries, HAVING clauses, CASE statements
  • Analytical thinking — translating clinical questions into structured queries
  • Risk classification logic — defining meaningful patient risk tiers from raw data
  • Query optimization and readability

Big Data · Sentiment Analysis

AI Relationships on Reddit — Sentiment Analysis

A big data analysis exploring how public sentiment toward AI relationships has evolved across Reddit communities.

The Goal

Investigate the big question: "How has sentiment towards relationships with AI changed over time?" by analyzing Reddit posts across multiple subreddits (ChatGPT, Replika, Grok, and others) using VADER sentiment analysis, BigQuery SQL, and Tableau visualizations.

What I Contributed

  • Queried large-scale Reddit datasets in Google BigQuery using SQL with REGEXP_CONTAINS for keyword filtering across AI-related subreddits
  • Applied VADER sentiment scoring to analyze compound sentiment across thousands of posts, comparing subreddits and time periods
  • Discovered key insight: sentiment dropped sharply in 2023 when Replika removed romantic conversation features, causing user backlash
  • Built Tableau visualizations showing sentiment trends, subreddit comparisons, and upvote ratio distributions
  • Analyzed engagement patterns — finding that AI companion-specific subreddits had near-perfect upvote ratios compared to general AI subreddits

Skills Gained

  • Big data querying with Google BigQuery SQL on large Reddit datasets
  • Natural Language Processing — VADER sentiment analysis at scale
  • Tableau dashboard design for communicating NLP findings
  • Research methodology — formulating big questions and sub-questions, then validating with data

National Parks Web Scraping & Search Engine

The Goal

Build an end-to-end information retrieval system: scrape content from NPS.gov, design a custom text analyzer, ingest documents into Elasticsearch, and execute advanced search queries including match, phrase match, wildcard, fuzzy, and boolean queries.

What I Contributed
